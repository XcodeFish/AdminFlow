# AdminFlow 前端文档

## 项目说明

AdminFlow 是一个基于Vue 3、Vite和Element Plus的现代化后台管理系统。本文档包含了项目的主要技术说明和开发指南。

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 主要技术栈

- **框架**: Vue 3 + TypeScript + Vite
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
